# 📚 Book Automation System

A Java desktop application developed to automate bookstore inventory and sales management using Java Swing and Object-Oriented Programming.

## 🚀 Features

- 📖 Manage book inventory
- 🔍 Search books
- 💰 Sales management
- 📊 Sales reports
- ⚠️ Low stock alerts
- 💾 File-based data storage

## 🛠 Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- File Handling

## 📂 Project Structure

```
src/
books.txt
sales.txt
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Run the main Java file.
