# 📚 Book Automation System

A Java Swing desktop application developed to automate bookstore inventory and sales management. The application provides an intuitive graphical user interface for managing books, processing sales, searching inventory, and generating reports.

---

## 🚀 Features

- 📖 Add, update, and manage book inventory
- 🔍 Search books by title and author
- 💰 Process book sales and generate receipts
- 📊 View sales statistics and reports
- ⚠️ Monitor low stock items
- 💾 File-based data storage using text files
- 🖥️ Interactive GUI built with Java Swing

---

## 🛠️ Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- File Handling
- VS Code

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/awwws0m/Book-Automation-System.git
```

2. Open the project in VS Code or any Java IDE.

3. Make sure Java JDK is installed.

4. Compile and run:

```text
Main.java
```
